Seattle Icons

Icons representing neighborhoods, landmarks, & other important features of Seattle, WA, released as Creative Commons.

    Use Seattle Icons in your:

  • Civic applications
  • Local blogs & other publications
  • Presentations about local issues
  • And wherever else you want!

What is this?

This project is just barely started! We held our first icon design-a-thon as part of CodeAcross Seattle, and upcoming events will expand the icon set. The goal is to create a resource of free-to-use icons that represent the city's neighborhoods, landmarks, issues, city services, & other important features.

How many icons exist so far?

One! You'll see it in the header above! It's the Space Needle.

However! At our first icon design-a-thon, participants sketched ideas for icons that you can use as the basis for SVG icons. Check out those sketches here.

Who works on this?

Seattle Icons is a project started at CodeAcross Seattle, an event focused on open data and civic technology organized by Code for Seattle.

Email Seth Vincent at svincent@codeforamerica.org if you're interested in getting involved!

How can I help?

Make an icon for your neighborhood, or for a landmark or issue in Seattle that interests you!

Attend a Code for Seattle meetup to get involved as an organizer of Seattle Icons. Join our meetup group to get notifications about upcoming icon design-a-thons.

You can also fork this repository on GitHub to help build the site, and to add icons (a simpler icon submission process is in the planning stages).
